API Sandbox
============

Fork this repo to start making an Express API :)

# Setup

Install packages:
```
yarn
```

Serve your app:
```
yarn serve
```

Visit app at default location:
```
http://localhost:3000
```

Lint your code:
```
yarn lint
```

Test your code:
```
yarn test
```

Lint, test and serve your code during development:
```
yarn watch
```

Create a `.env` file to override defaults (requires app restart):
```
cp example.env .env
```
