API Sandbox
============

Fork this repo to start making an Express API :)

# Setup

Install packages:
```sh
yarn
cd /var/www/shop
```

Serve your app:
```
yarn serve
```

Visit app at default location:
```
http://localhost:3000
```

Lint your code:
```
yarn lint
```

Watch and test your code:
```
yarn test --watch
```

Develop your code:
```
yarn watch
```

Create `.env` file to override defaults (requires app restart):
```
cp example.env .env
```
